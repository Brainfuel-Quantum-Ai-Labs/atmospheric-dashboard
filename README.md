# Atmospheric Gas Monitoring Dashboard

Real‑time web application for retrieving and visualizing atmospheric gas concentrations (CO₂, CH₄, H₂O) from ground‑based long‑path absorption spectroscopy.

## Features
- Live gas concentration updates
- Interactive time‑series charts
- Alert thresholds for abnormal levels
- Historical data analysis

## Installation
1. Clone the repository
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `source venv/bin/activate` (Windows: `venv\Scripts\activate`)
4. Install dependencies: `pip install -r requirements.txt`
5. Run the app: `python app.py`
6. Open http://127.0.0.1:5000 in your browser
